Changelog

New features, fixes, and shipped intent. Tracked release-by-release.

VuePoint
v1.3.3
May 2026
fix: CAGR annualization · health score · integrity gate
  • fix: CAGR formula produced runaway values on short holds (e.g. 5,441% at 19 days, -100% at 2 days) — corrected to true annualized return; requires ≥1yr hold to calculate
  • add: sub-annual guard — positions <1yr show <1yr; positions <30 days show —
  • fix: null-input hardening — missing purchase date, zero basis, or zero market value now render instead of formula errors
  • add: portfolio health score — A–F grade derived from multi-factor scoring algorithm (gain, drift, signal, position weight)
  • add: integrity check integration — validates integrity token; flags tampered or unlicensed copies
v1.3.2
May 2026
add: setup wizard · completeness indicators · error handling
  • add: first-run setup wizard — prompts account names + drift threshold on first open; re-runnable from Settings & Tools menu
  • add: per-position completeness indicator — green/amber/red dot per row based on required field fill state
  • add: portfolio-level completeness % displayed in myPortfolio header row
  • add: formula overwrite guard — onEdit detects protected cell writes, surfaces undo instructions via toast
  • fix: menu actions were failing silently on error — all actions now surface actionable error messages
  • fix: read-only tabs showing misleading "changes not saved" notice — removed
v1.3.1
May 2026
add: drift bars · row tinting · integrity toast · timestamp
  • add: drift bar in col N — inline block-bar renders alongside drift % for faster visual scanning
  • add: full-row gain/loss tint — winning rows shade green, losing rows shade red across all columns
  • change: ticker cells restyled — symbols render as cyan pill against dark background
  • add: onOpen integrity toast — green confirmation shown each time file opens clean
  • add: last-refreshed timestamp injected into myPortfolio header on each price refresh
v1.3.0
Apr 2026
change: dark theme · signal pills · LTCG states · header bands
  • change: myPortfolio theme overhauled — dark navy base, cyan accents, restructured visual hierarchy
  • change: signal column — plain text replaced with colored pill badges (BUY/TRIM/HOLD/REVIEW)
  • add: LTCG pill states — Clear / Soon / Pending per position for at-a-glance tax-lot status
  • change: account header rows — darker navy band applied to each account block for cleaner separation
  • change: column headers — cyan uppercase styling on dark card background
v1.2
Mar 2026
add: signal col · CAGR · drift CF · ticker DB expansion
  • add: signal column — TRIM/BUY/HOLD/REVIEW derived from drift % vs threshold and gain state
  • add: CAGR column — annualized return per position calculated from purchase date
  • add: drift conditional formatting — amber (over threshold) / cyan (under) / green (in range)
  • add: signal conditional formatting — color-coded per action type for instant scan
  • fix: LTCG date formula showing 365 on empty rows — resolved
  • fix: unknown ticker on onEdit produced silent blank — now surfaces toast notification
  • change: ticker DB expanded ~60 → 300+ pre-mapped entries
  • change: gridlines removed across all tabs

  • add: conditional formatting on Gain $ and Gain % columns — green/red by value
  • add: Settings tab — configurable drift threshold, cash drag threshold, LTCG warning days, account names
