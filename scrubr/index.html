Questions

Good to know.

Does anything ever upload? ▼
No. Every operation, decode, inspect, redact, annotate, export, runs in your browser via canvas. The only network call Scrubr ever makes is a Pro license check, which sends a license key, never file content. "We never see your files" is a literal claim.
What's free and what needs Pro? ▼
Inspect is always free, along with basic annotate tools (arrow, box, text) on a single file. Pro unlocks real redaction (black out, blur, pixelate), the full annotate set, batch files, and multi-page PDF export, a one-time purchase whose single license key unlocks everything.
Is Scrubr Pro a subscription? ▼
No. Scrubr Pro is a one-time purchase. One license key unlocks the full toolkit with no subscription and no renewal.
Does redaction actually remove the data, or just cover it? ▼
It's removed for real. Redacted regions are rasterized into the exported file, so the content underneath is physically gone, not layered over. This is different from tools that just draw a box on top and leave the original data intact underneath.
Does it work on PDFs? ▼
Yes. PDF pages are rendered to canvas and treated like images, then reassembled into a single multi-page PDF on export. One honest note: exported PDF pages are flattened images, not searchable text. True text-layer redaction is on the roadmap.
